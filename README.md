# Singapore Family Trip 2026

Mobile-first web app for a 2026 Singapore family itinerary.

## Preview

Open `index.html` directly, or run a local static server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173/`.

## Contents

- 5-day family itinerary for 2026/8/15-2026/8/19
- Day-by-day timeline
- Official attraction links
- Ticket, packing, and pacing notes
- Mobile-first responsive layout
